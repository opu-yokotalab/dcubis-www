FREE WALKERは、360°のパノラマ写真と、道なりに撮影した連続写真の組み合わせにより、仮想空間を構築したものです。臨場感があり、自由に散策可能で、軽快に動作可能で、操作も容易です。

※FREE WALKER は、Javaアプレットを使用しています。起動するには、Java Plug-in が必要です。
Sun のホームページから Java Plug-in をダウンロードできます。
1. Sun の Java ダウンロードページを開きます。
2. 「今すぐダウンロード。無料！」をクリックしてダウンロードします。
